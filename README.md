# Ad Monitor

Monitor de anuncios con **50 banners visibles** y **50 ventanas ocultas** con Smartlink directo.

## Características

- 50 iframes de banners visibles rotando todos los formatos (160x300, 160x600, 300x250, 320x50, 468x60, 728x90, Native, Popunder, SocialBar)
- 50 ventanas ocultas enviando tráfico real al Smartlink
- Clics reales via MouseEvent + postMessage cada 7 segundos
- Conteo de impresiones y CTR en tiempo real
- Recarga automática cada 30 segundos
- VPN-ready: la IP del navegador determina el país detectado por Adsterra

## Uso

Abre [https://NEXYVLYS.github.io/ad-monitor](https://NEXYVLYS.github.io/ad-monitor) y presiona **INICIAR**.

No requiere instalación — es una página HTML pura.
